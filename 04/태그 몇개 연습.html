(여기는 h1태그)태그들 연습

1.p태그는

단락 자체를 나눈다

2.br은
그냥 줄이 바뀐 것처럼 화면에 보여줄 뿐,
실제로 단락이 나뉜 건 아니다.

(그래서 css 사용해 단락 스타일 만들 때 문제 생길 수 있으므로 p를 이용하는 게 더 나음.)

3.blockquote=인용문:글을 약간 들여씀

4.strong 태그=텍스트를 굵게 표시함

5.b태그도 텍스트 굵게 표시함

6.em이나 7.i 태그는 택스트 기울이기.이탤릭채.

8.abbr태그:줄임말 입력 후 마우스 커서 올려놓으면 설명이 뜬다.

9.본문 글이나 이미지 등이 있고 여기에 cite 태그를 쓰면 뭔가 설명, 출처를 붙이게 된다.

10.code 소스 코드 입력용

11.small태그는 글씨 더 작게 하는 것원래 크기는 이 정도.

12.sup태그:윗첨자윗첨자

13s태그:취소선

14u태그:밑줄

15 ins태그:웹페이지의 새로 추가된 텍스트에 밑줄을 표시하는 경우에 사용

16 del태그:웹페이지의 기존 텍스트에서 삭제/취소하고 싶은 내용에 취소선 표시하는 경우에 사용
